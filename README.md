📌 About The Project

This project is a Student Management System written in C, designed to demonstrate:

- 📁 File Handling
- 🧱 Structured Programming
- ⚙️ Modular Code Design
- 🖥️ CLI-based Interaction

It allows users to store, retrieve, and manage student data efficiently.

---

✨ Features

Feature| Description
➕ Add Student| Store student details
📋 View Students| Display all records
🔍 Search Student| Find by ID
🗑️ Delete Student| Remove record
💾 Persistent Storage| Data saved using file

---

🛠 Tech Stack

- 💻 C Programming
- 📂 File Handling (".dat")
- ⚙️ Makefile
- 📱 Termux
