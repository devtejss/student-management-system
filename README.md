📌 Overview

A command-line based Student Management System built using C language.
This project demonstrates core concepts like structures, file handling, and modular programming.

---

🚀 Features

✨ Add Student
📋 View All Students
🔍 Search Student by ID
🗑️ Delete Student
💾 Persistent Storage using File Handling

---

🛠 Tech Stack

- 💻 C Programming Language
- 📁 File Handling (".dat" files)
- ⚙️ Makefile (Build Automation)
- 📱 Termux (Development Environment)

---

📁 Project Structure

student-management-system/
├── src/
│   ├── main.c
│   ├── student.c
├── include/
│   ├── student.h
├── data/
│   ├── students.dat
├── Makefile
└── README.md

---

▶️ How to Run

# Compile
make

# Run
./app

# Clean build files
make clean
